WebPerformance Schulung

Eine gute WebPerformance wird alle immer wichtiger. Schnell ladbare Webseiten bekommen ein besseres Google-Ranking und führen zu einer besseren Conversion-Rate. Als angenehmen Seiteneffekt benötigt man bei gleicher Last weniger Webserver.

Haben Sie eine Frage zur Schulung? Dann schreiben Sie mir bitte eine E-Mail an {{ site.email }} oder rufen Sie mich an (Tel. {{ site.phone_number }}).

In meiner 1-tägigen Schulung behandele ich mit Ihnen die folgende Themen:

TCP

  • TCP Grundlagen
  • Slow-Start

HTTP/1.1

  • Grundlagen
  • Komprimierung

HTTP/2

  • TLS
  • Header Optimierung

HTML, CSS und JavaScript

  • technische Hintergründe (warum lädt was wann)
  • Wasserfallanalyse
  • Optimierung
  • Komprimierung

Server-Software

  • nginx
  • Apache
  • Varnish

Bildoptimierung

  • Unterschiedliche Formate
  • Einsatz von externen APIs
  • Responsiveness

Caching

  • HTTP-Caching
  • Page-Caching
  • Fragment-Caching
  • CDNs

Was muss mitgebracht werden?

Jeder Teilnehmer muss einen Laptop mit einem installierten aktuellen Chrome Browser und funktionierendem WLAN mitbringen.
